Dự án app chat online (MyChat)
Các tính năng và công nghệ đi kèm (dự kiến):
	
	+ Giao diện đồ họa: Terminal + PyQt6
	
	+ Database: SQlite3(Data), NoSQL - Dictionary Python(cache)

	+ Kết nối server-client: Websocket

	+ Gửi file hình ảnh và âm thanh: Websocket + FastAPI

	+ Nhắn tin giữa các người dùng: thông qua server với Websocket

	+ Gọi điện: Pyaudio + UDP protocol

	+ Group Chat: cập nhật sau

	+ Tìm kiếm người dùng khác và danh sách bạn bè: cập nhật sau

	+ Đăng kí và đăng nhập: sử dụng database để lưu (có hashing mật khẩu)
